#### Исследовательский анализ данных (финансирование инновационных предприятий: объёмы, типы и доходность)
Exploratory Data Analysis and visualization (historical dataset of the venture ecosystem)

**Цель исследования** - осуществлить углублённый анализ исторических данных о стартапах и выработать рекомендации по выбору оптимальной бизнес‑модели. **Основной задачей** является исследовательский анализ данных о стартапах с последующим формированием обоснованных выводов по результатам исследования

**Структура работы**

python

- проведена предобработка данных;
  - проверка корректности, знакомство,
  - работа с типами данных и анализ пропусков   
- выполнено выборочное исследование по запросу компании-заказчика;
  - проанализированы динамика финансирования и образовательных характеристик сотрудников,
  - дополнительный обзор загруженных данных
- выполнен исследовательский анализ данных объединенного датасета;
  - проведена интеграция данных,
  - выявление выбросов и изучение ценовой динамики и типичного числа раундов финансирования в разрезе статусов стартапов
- на основе полученных результатов, сформулированы обоснованные рекомендации для финансовой компании‑заказчика
