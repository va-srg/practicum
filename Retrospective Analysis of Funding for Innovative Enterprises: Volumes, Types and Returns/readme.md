#### [Исследовательский анализ данных (финансирование инновационных предприятий: объёмы, типы и доходность)](https://github.com/va-srg/practicum/blob/fd2c0c1b8d87aa51a52fd3c36e7dc8c1cb3f2022/Retrospective%20Analysis%20of%20Funding%20for%20Innovative%20Enterprises%3A%20Volumes%2C%20Types%20and%20Returns/Exploratory%20Data%20Analysis%20and%20visualization%20(historical%20dataset%20of%20the%20venture%20ecosystem).ipynb)
<span style="color: #c0c0c0;">Exploratory Data Analysis and visualization (historical dataset of the venture ecosystem)</span>

**Цель исследования** - осуществлить углублённый анализ исторических данных о стартапах и выработать рекомендации по выбору оптимальной бизнес‑модели. **Основной задачей** является исследовательский анализ данных о стартапах с последующим формированием обоснованных выводов по результатам исследования

В данном проекте: 
- проведена предобработка датасетов;
- выборочное исследование динамики финансирования и образовательных характеристик сотрудников, интеграцию данных, выявление выбросов и изучение ценовой динамики и типичного числа раундов финансирования в разрезе статусов стартапов;
- на основе полученных результатов, сформулированы обоснованные рекомендации для финансовой компании‑заказчика

Структура работы:
1. Предобработка данных: загрузка и преобразование
2. Выборочное исследование по запросу компании-заказчика
3. Исследовательский анализ данных объединенного датасета
4. Итоговый вывод и рекомендации
