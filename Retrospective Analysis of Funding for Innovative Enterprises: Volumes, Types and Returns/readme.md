#### Исторический исследовательский анализ данных финансирования инновационных предприятий: объёмы, типы и доходность
<span style="color: #c0c0c0;">Exploratory Data Analysis and visualization (historical dataset of the venture ecosystem)</span>

**Цель исследования** - осуществлить углублённый анализ исторических данных о стартапах и выработать рекомендации по выбору оптимальной бизнес‑модели. **Основной задачей** является исследовательский анализ данных о стартапах с последующим формированием обоснованных выводов по результатам исследования

В данном проекте: 
- проведена предобработка датасетов;
- исследование динамики финансирования и образовательных характеристик участников, интеграцию данных, выявление выбросов и символических сделок, а также изучение ценовой динамики и типичного числа раундов финансирования в разрезе статусов стартапов;
- на основе полученных результатов, сформулированы обоснованные рекомендации для финансовой компании‑заказчика

Структура работы:
1. Знакомство с данными: загрузка и первичная предобработка
2. Предобработка данных, выборочное исследование исторических данных
3. Исследовательский анализ объединённого датасета
4. Итоговый вывод и рекомендации
