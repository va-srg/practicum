#### [Анализ продаж курсов образовательной платформы](https://datalens.yandex/vwu43dcirxzif)
Course sales analysis for the educational platform

**Цель проекта** - создать интерактивный дашборд для мониторинга и анализа продаж курсов образовательной платформы (включая географию спроса и популярность). **Основная задача** - реализовать все требования заказчика для своевременной корректировки бизнес‑процессов на основе данных о продажах и геоданных

**Структура работы**

excel/sheets

- сформированы 5 вспомогательных таблиц с использованием формул DAX, включая генерацию случайных чисел из заданного диапазона

python

- предобработаны и сгенерированы 500 000 строк (из вспомогательных таблиц, подготовленных ранее) для полноценной имитации структуры БД (postgresql);
- итоговый датасет включающий 6 датафреймов экспортирован в excel файл для дальнейшей работы с ним в datalens

datalens

- дашборд состоит из 4 вкладок (включая подробное описание метрик, чартов, селекторов);
- все необходимые подробности по содержанию расписаны в аннотации;
- дополнительно приведена аналитическая записка
