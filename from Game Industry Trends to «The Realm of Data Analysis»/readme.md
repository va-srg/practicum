#### Предобработка данных (команда игры «Мир анализа данных»)

<span style="color: #c0c0c0;">Data Preprocessing (Team of the Game — The Realm of Data Analysis)</span>

**Цель исследования** - анализ развития игровой индустрии начала XXI века для создания статьи, направленной на привлечение новой аудитории к игре. **Основной задачей** является предобработка собранных данных командой игры. 

В данном проекте:
- изучены исходные данные,
- проверена их корректность,
- проведена предобработка для дальнейшей выгрузки результата и подготовки на основании данных статьи

Структура работы:
  1. Загрузка и знакомство с данными
  2. Проверка ошибок в данных и их предобработка
  3. Фильтрация данных
  4. Категоризация данных
