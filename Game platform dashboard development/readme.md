#### [Тренды на рынке игр](https://github.com/va-srg/practicum/tree/main/Game%20platform%20dashboard%20development/dashboard%20overview)

Game market trend analysis

**Цель проекта** - создать интерактивный дашборд в Apache Superset для мониторинга трендов на рынке игр и комплексного анализа ключевых метрик (цен, установок, вовлеченности, релизов) на основе исторических данных. Дашборд должен помочь при принятии решений по ценовой политике, планированию релизов и выявлению точек роста. **Основная задача** - реализовать разделы анализа игроков, игр, игровых достижений и стоимости продукта с визуализацией ключевых метрик и настройкой кросс‑фильтрации с предустановленными значениями периода и гранулярностью по месяцам, включая специальный чарт year-to-year анализа количества игроков

**Структура работы**

sql

- сформирован объединенный датасет для дальнейшего использования на уровне BI-инструмента на основе таблиц БД (postgresql) - структура которых представлена в er-диаграмме;
- сформирован дополнительный виртуальный датасет с последовательностью месяцев на уровне BI-инструмента для реализации сравнения в чарте год-к-году с использованием jinja

superset
- созданы виртуальные датасеты на основе подготовленных запросов, также созданы кастомные метрики расчетов на уровне датасета;
- разработана четырехстраничная структура дашборда, с дополнительной вкладкой аннотации внутри него (включая обучение новых пользователей);
- настроены фильтры, их значения по умолчанию, продумана и реализована кросс-фильтрация;
- реализован чарт сравнения в чарте год-к-году с использованием jinja
