### Раздел 1 - Заголовок
---
**Название отчета:** "Тренды на рынке игр"

**Бизнес-цель отчета:** отслеживать тренды на рынке игр, анализировать ключевые показатели: игровое поведение пользователей, популярность игр и изменения цен

**Подразделение заказчика:** топ-менеджмент; игровой департамент; аналитический департамент

**Заказчик:** Компания-разработчик игр

**Целевая аудитория дашборда:** топ-менеджеры - для оценки потенциала новых релизов и корректировки ценовой политики; аналики компании - для анализа эффективности отдельных игр и выявления точек роста; специалисты игрового департамента - сотрудники, занимающиеся разработкой и выпуском игровых продуктов, которым нужен инструмент для оперативного мониторинга рынка

**Дата обновления:** 24 октября 2025 года

**Автор:** Васильев Сергей

**Версия	1.1:** (значение фильтра период при первом включении настроено с 2024-06-01 по 2025-01-01)
