Цель проекта - провести комплексный анализ рынка общественного питания города для поддержки стратегического решения частных инвесторов по входу на рынок — определить перспективные локации, форматы заведений и ценовые сегменты для открытия нового объекта

По результатам проведённого исследовательского анализа данных сформулированы итоговый вывод и рекомендации для заказчика
