#### [Исследовательский анализ данных (открытие заведения общественного питания)](https://github.com/va-srg/practicum/blob/main/«We’re%20Open»:%20new%20catering%20establishment/Exploratory%20Data%20Analysis%20and%20visualization%20(for%20private%20investors).ipynb)

<span style="color: #c0c0c0;">We’re Open»: new catering establishment</span>

**Цель исследования** - провести комплексный анализ рынка общественного питания города для поддержки стратегического решения частных инвесторов по входу на рынок — определить перспективные локации, форматы заведений и ценовые сегменты для открытия нового объекта. **Основной задачей** является полный исследовательский анализ данных для частных инвесторов, планирующих открытие нового заведения в n-городе

В данном проекте:
- изучены исходные данные (проверена их корректность, проведена предобработка);
- выполнен многоаспектный исследовательский анализ (оценены объёмы и динамика рынка, проведена сегментация по форматам заведений и ценовым категориям, проанализирована географическая дистрибуция точек, изучены конкурентная среда);
- на основе полученных инсайтов сформулированы обоснованные рекомендации по выбору формата, локации, ценовой политики и маркетинговых инструментов для успешного входа на рынок общественного питания n‑города

Структура работы:
1. Загрузка и знакомство с данными
2. Предобработка данных
3. Исследовательский анализ данных
4. Итоговый вывод и рекомендации
