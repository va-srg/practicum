Exploratory Data Analysis and visualization (for private investors)

**Цель исследования** - провести комплексный анализ рынка общественного питания города для частных инвесторов по входу на рынок - определить перспективные локации, форматы заведений и ценовые сегменты для открытия нового объекта. **Основной задачей** является исследовательский анализ данных для частных инвесторов, планирующих открытие нового заведения в n-городе

**Структура работы**

python

- изучены исходные данные;

  - проверена их корректность
- выполнена предобработка данных;
  - работа с пропусками,
  - нормализация,
  - оптимизированы типы
- выполнен многоаспектный исследовательский анализ (оценены объёмы и динамика рынка, проведена сегментация по форматам заведений и ценовым категориям, проанализирована географическая дистрибуция точек, изучены конкурентная среда);
- на основе полученных инсайтов сформулированы обоснованные рекомендации по выбору формата, локации, ценовой политики и маркетинговых инструментов для успешного входа на рынок общественного питания n‑города
