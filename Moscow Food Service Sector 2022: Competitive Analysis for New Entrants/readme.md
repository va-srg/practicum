**Цель проекта** - провести комплексный анализ рынка общественного питания города для поддержки стратегического решения инвесторов фонда по входу на рынок — определить перспективные локации, форматы заведений и ценовые сегменты для открытия нового объекта (кафе/ресторана/бара)
По результатам проведённого исследовательского анализа данных сформулированы итоговый вывод и рекомендации для заказчика:
  1. Общий обзор проделанной работы
  2. Ответы на исследовательские вопросы, или главные выводы
  3. Рекомендации на основе анализа данных
