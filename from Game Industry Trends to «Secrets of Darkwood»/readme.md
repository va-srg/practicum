**Цель исследования** - анализ развития игровой индустрии начала XXI века для создания статьи, направленной на привлечение новой аудитории к игре. **Основной задачей** является предобработка собранных данных командой игры *«Секреты Темнолесья»*. В данном проекте:
- изучим исходные данные,
- проверим их корректность,
- проведем предобработку для дальнейшей выгрузки результата
