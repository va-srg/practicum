**Цель исследования** - анализ развития игровой индустрии начала XXI века для создания статьи, направленной на привлечение новой аудитории к игре. **Основной задачей** является предобработка собранных данных командой игры *«Секреты Темнолесья»*. 

В данном проекте:
- изучены исходные данные,
- проверена их корректность,
- проведена предобработка для дальнейшей выгрузки результата и подготовки на основании данных статьи
